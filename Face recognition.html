FACE RECOGNITION DOOR UNLOCK SYSTEM

Objective:

Face Recognition–Based Door Lock System, a smart and contactless security solution. In sensitive places like examination rooms, it is very important to allow entry only to authorized persons. Traditional locks and ID cards can be lost, shared, or misused. To overcome these problems, we designed a system that unlocks the door only when a registered face is recognized. This improves security and avoids unauthorized entry.”
